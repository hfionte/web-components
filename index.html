Web Components

http://webcomponents.org/

01: Template Element

http://www.w3.org/TR/html5/scripting-1.html#the-template-element

The template element is used to declare fragments of HTML that can be cloned and inserted in the document by script.

In a rendering, the template element represents nothing.

template.content returns the contents of the template.

02: HTML Imports

http://w3c.github.io/webcomponents/spec/imports/

HTML Imports are a way to include and reuse HTML documents in other HTML documents.

<link rel="import" href="/imports/heart.html">

03: Shadow DOM

http://www.w3.org/TR/shadow-dom/

The Shadow DOM is a subtree of DOM elements that are rendered with the main document, but are not part of the main DOM tree. An element that contains a subtree is called the "shadow host". The root node of the sub tree is the "shadow root".

You can view the shadow DOM elements of existing elements such as video and input, or you can create your own shadow DOM using createShadowRoot().

04: Custom Element

http://w3c.github.io/webcomponents/spec/custom/

Custom elements provide a way for web developers to build their own, fully-featured DOM elements. Though it was long possible to create DOM elements with any tag names in HTML, these elements weren't very functional.

The name of a custom element must be all lowercase, must contain at least one hyphen, and can't be one of the already existing names with hyphens.

Custom elements are registered with document.registerElement()

05: Polymer

https://www.polymer-project.org/1.0/

The Polymer library is designed to make it easier and faster for developers to create great, reusable components for the modern web.

A catalog of Polymer elements is available at https://elements.polymer-project.org/ and there are more at https://customelements.io/.

06: Vulcanize

Vulcanization is a process for converting polymers into more durable materials. (paraphrased from wikipedia)

Vulcanizing Polymer components prepares them for use in production by compiling all scripts and imports into one page.
